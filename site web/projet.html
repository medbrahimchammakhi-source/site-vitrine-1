1. Définitions

Site Web : Ensemble de documents et de ressources reliés par des liens hypertextes.

HTML : Langage de balisage utilisé pour structurer le contenu d'une page web.

CSS : Langage utilisé pour définir la présentation et le style des documents HTML.
